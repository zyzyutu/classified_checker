# 涉密信息综合检查系统

## 项目简介
本项目是一个基于Python开发的本地涉密信息检查工具，支持对多种文件类型进行关键词扫描，包含以下功能：
- 文本、Word、Excel文档的涉密关键词检测
- 网页内容爬取与检查
- 本地图片OCR文字识别与检查（支持批量图片遍历）
- 自动生成结构化检查报告，包含文件路径、检测结果统计

## 使用说明
1.  运行 `main.py` 启动程序
2.  配置文档目录、图片目录、关键词列表
3.  点击「开始检查」执行扫描
4.  检查完成后自动生成报告文件

## 开发环境
- Python 3.11
- PyCharm
- 依赖库：pytesseract, pillow, requests, pandas
